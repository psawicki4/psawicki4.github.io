### Tech stack
- **Framework**: Angular 21
- **UI**: Angular Material, Tailwind CSS 4, Sass
- **State**: NgRx Signals
- **I18n**: Transloco
- **3D**: three.js
- **Data grid**: AG Grid
- **Date/Time**: dayjs, luxon (Material adapter)
