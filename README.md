### Tech stack
- **Framework**: Angular 20
- **UI**: Angular Material, Tailwind CSS 4, Sass
- **State/Signals**: NgRx Signals
- **I18n**: @ngx-translate
- **3D**: three.js
- **Data grid**: AG Grid
- **Date/Time**: dayjs, luxon (Material adapter)
